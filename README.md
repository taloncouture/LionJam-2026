# About
This is my submission for LionJam and the winner of the community vote! LionJam is the yearly game jam at Columbia University. My game is called **Great Pyramid of Pizza** where you are instructed by
King Pizza to build a great pyramid. This is turn-based and tycoon-like, where you place down tiles on an island to produce pizza.

This is all made in Python using the Pygame library.

# How to play
The quickest way would be to go over to my page on Itch.io, but otherwise you can do:
```
pip install pygame
pip install pyinstaller
pyinstaller --onefile main.py --add-data "assets:assets"
```
